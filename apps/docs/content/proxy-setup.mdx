# Proxy Setup

Proxy settings apply to `apps/agent`.

Set:

- `PROXY`

Supported proxy formats:

- `host:port`
- `http://host:port`
- `https://host:port`
- `socks5://host:port`
- `http://username:password@host:port`

Credentials should be supplied directly in `PROXY` when required.

## Operational Guidance

- Keep proxy credentials out of source control.
- Track provider-specific failure rates.
- Increase `AGENT_WORKER_CONCURRENCY` only after proxy pool stability is verified.
