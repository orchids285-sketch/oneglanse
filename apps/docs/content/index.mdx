# Introduction

OneGlanse helps teams understand how brands appear in real LLM interfaces, not
just API responses.

It runs prompts across supported providers, extracts responses and sources, and
turns them into visibility, sentiment, and citation intelligence.

## What It Tracks

- brand presence by provider
- competitive co-mentions
- recommendation quality and sentiment
- citation sources and source concentration

## What Makes It Different

OneGlanse focuses on UI-level collection with real browser sessions. That gives
you a closer view of what users actually see in products like ChatGPT, Gemini,
Perplexity, Claude, and Google AI Overview.

## Core Ideas

- local-first auth capture
- portable auth bundles for VPS sync
- persistent Camoufox runtime profiles for stable repeated runs
- split app and public Docker stacks for safer VPS redeploys
- separate operational and analytical data stores
